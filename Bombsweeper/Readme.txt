## Instructions for Running BombSquare:

### 1. Compile the Code:
   - Open a terminal or command prompt.
   - Navigate to the directory containing the Java files.
	javac Driver.java
     ```bash
     javac Driver.java
     ```

### 2. Run the Program:
   - After successful compilation, run the program with the following command:
	javac Driver.java
     ```bash
     java Driver
     ```

### 3. Input Board Size:
   - Follow the prompts to enter the width and height of the game board.
   - Example:
     ```
     Enter the width of the game board: [Your Width]
     Enter the height of the game board: [Your Height]
     ```

### 4. Play the Game:
   - The Minesweeper game will start with the specified board size.
   - Click on the squares to uncover bombs and numbers.
